# Free Bet Blackjack (scaffold)

This repository is a minimal scaffold for a Free Bet Blackjack web app (TypeScript + React + Vite).

Quick start (Windows):

1. Install Node.js LTS from https://nodejs.org/en/ or via winget:

```powershell
winget install OpenJS.NodeJS.LTS
```

2. From project root:

```powershell
cd freebet-blackjack
npm install
npm run dev
```

3. Run tests:

```powershell
npm run test
```

Notes:
- This scaffold includes a small engine module in `src/engine` and a basic UI in `src/App.tsx`.
- If you want, I can continue by implementing the full rule engine, unit tests for every edge case, and pixel-art assets.
